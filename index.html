CoilCalc — Web edition

Greetings from Seoul. MR-WPT coil calculator running entirely in your browser — no server, no backend, no data leaves your machine. The same CoilCalc physics core (Python module coilcalc) that powers the desktop and Electron builds is shipped to you as a pure-Python wheel and executed inside the page by Pyodide (CPython compiled to WebAssembly). It works offline after the first load, runs from any static host (GitHub Pages, Netlify, S3, even file://), and validated 26 / 26 analytical theory checks against Nagaoka, Mohan, Wheeler, Maxwell, Bessel-skin and Neumann reference results — all 41 pytest cases pass too.

Geometries: circular / polygonal / rectangular spiral · solenoid · conical · multi-layer · DD pad.
Conductors: solid round (Bessel) · Litz (Sullivan) · foil (Dowell) · PCB trace (Yue).
Physics: Mohan / Wheeler-Nagaoka / Zhao / universal Neumann + GMR for L · Maxwell coaxial + Neumann for M · Biot-Savart proximity for Q · Medhurst + GKMR + Yun for SRF · full Z-matrix N-coil solver for η.
